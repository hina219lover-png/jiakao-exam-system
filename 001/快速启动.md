# 快速启动指南

## 方式一：使用 IDEA 运行（推荐）

### 1. 用 IDEA 打开项目
1. 打开 IntelliJ IDEA
2. 选择 `File` -> `Open`
3. 选择项目文件夹：`d:\Study\ideaIU\xiangmu\zonghexiangmu\001`
4. 点击 `OK`

### 2. 配置数据库
在 IDEA 中打开文件：`src/main/resources/application.properties`

修改以下配置（根据你的 MySQL 配置）：
```properties
spring.datasource.username=root
spring.datasource.password=你的 MySQL 密码
```

### 3. 下载依赖
IDEA 会自动识别 Maven 项目并下载依赖。如果没有自动下载：
- 右键点击 `pom.xml`
- 选择 `Maven` -> `Reload Project`

### 4. 启动项目
**方法 1**: 直接运行主类
- 找到文件：`src/main/java/com/driving/DrivingTestApplication.java`
- 右键点击该文件
- 选择 `Run 'DrivingTestApplication.main()'`

**方法 2**: 使用 IDEA 的 Run Configuration
- 点击右上角的 `Add Configuration...`
- 点击 `+` 号，选择 `Spring Boot`
- 主类选择：`com.driving.DrivingTestApplication`
- 点击 `Apply` -> `OK`
- 点击绿色的运行按钮

### 5. 访问系统
启动成功后（看到"驾考系统启动成功！"提示），在浏览器访问：
- http://localhost:8080/login.html

## 方式二：使用命令行运行

### 前提条件
需要安装：
- JDK 17+
- Maven 3.6+
- MySQL 8.0+

### 1. 安装 Maven（如果未安装）
Windows 用户可以使用 Chocolatey 安装：
```bash
choco install maven
```

或手动下载安装：
1. 访问 https://maven.apache.org/download.cgi
2. 下载最新版本的 Maven
3. 解压到本地，例如：`C:\Program Files\Maven`
4. 添加环境变量：
   - MAVEN_HOME = C:\Program Files\Maven
   - Path 中添加：%MAVEN_HOME%\bin

### 2. 配置数据库
编辑 `src/main/resources/application.properties`，修改数据库连接信息

### 3. 下载依赖并编译
在项目根目录执行：
```bash
mvn clean install -DskipTests
```

### 4. 启动项目
```bash
mvn spring-boot:run
```

### 5. 访问系统
启动成功后访问：http://localhost:8080/login.html

## 常见问题

### Q1: IDEA 无法识别 Maven 项目？
**解决方法**: 
- 右键点击 `pom.xml` -> `Add as Maven Project`

### Q2: 缺少 Lombok 插件导致报错？
**解决方法**:
- File -> Settings -> Plugins
- 搜索 `Lombok` 并安装
- 重启 IDEA
- 开启注解处理：File -> Settings -> Build, Execution, Deployment -> Compiler -> Annotation Processors -> 勾选 `Enable annotation processing`

### Q3: 数据库连接失败？
**解决方法**:
1. 检查 MySQL 服务是否启动
2. 确认用户名和密码正确
3. 确保 MySQL 版本是 8.0+

### Q4: 端口 8080 已被占用？
**解决方法**:
编辑 `application.properties`，修改端口：
```properties
server.port=8081
```

### Q5: 前端页面空白或无法加载？
**解决方法**:
1. 确保后端服务已启动
2. 检查浏览器控制台是否有错误
3. 清除浏览器缓存后重试
