## 📦 快速实现模板

我为你准备了每个模块的代码模板。以下是使用示例：

### Service 模板
```java
@Service
public class QuestionService extends ServiceImpl<QuestionMapper, Question> 
    implements IQuestionService {
    
    @Override
    public IPage<Question> getQuestionsByChapter(Integer chapter, Page page) {
        return this.page(page, new LambdaQueryWrapper<Question>()
            .eq(Question::getChapter, chapter)
            .orderByAsc(Question::getId));
    }
}
```

### Controller 模板
```java
@RestController
@RequestMapping("/api/question")
@RequiredArgsConstructor
public class QuestionController {
    
    private final QuestionService questionService;
    
    @GetMapping("/list")
    public Result<IPage<Question>> list(
        @RequestParam(required = false) Integer chapter,
        @RequestParam(defaultValue = "1") Integer current,
        @RequestParam(defaultValue = "20") Integer size
    ) {
        Page page = new Page(current, size);
        IPage<Question> result = questionService.getQuestionsByChapter(chapter, page);
        return Result.success(result);
    }
}
```
