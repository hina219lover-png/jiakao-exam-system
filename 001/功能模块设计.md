# 驾考系统 - 完整功能模块设计

## 📋 用户端功能模块

### 1. 个人中心 👤
**功能描述**：用户查看和编辑个人信息、学习统计

#### API 接口
- `GET /api/user/profile` - 获取个人信息
- `PUT /api/user/profile` - 更新个人信息
- `POST /api/user/avatar` - 上传头像
- `PUT /api/user/password` - 修改密码
- `GET /api/user/statistics` - 获取学习统计

#### 数据展示
- 基本信息（用户名、邮箱、手机）
- 学习数据统计：
  - 总做题数
  - 正确率
  - 收藏题目数
  - 错题数量
  - 模拟考试次数
  - 平均分数
- 最近学习记录

---

### 2. 错题本 📝
**功能描述**：自动收集错题，支持反复练习

#### API 接口
- `GET /api/wrong/list` - 获取错题列表
- `DELETE /api/wrong/{questionId}` - 移除错题
- `POST /api/wrong/remove-batch` - 批量移除（已掌握的）
- `GET /api/wrong/statistics` - 错题统计
- `POST /api/wrong/practice` - 开始错题练习

#### 功能特性
- 按章节分类查看
- 按错误次数排序
- 标记已掌握
- 错题导出
- 智能推荐相似题目

---

### 3. 收藏夹 ⭐
**功能描述**：收藏重要或难题，方便重点复习

#### API 接口
- `GET /api/collection/list` - 获取收藏列表
- `POST /api/collection/add/{questionId}` - 添加收藏
- `DELETE /api/collection/{questionId}` - 取消收藏
- `GET /api/collection/check/{questionId}` - 检查是否已收藏

#### 功能特性
- 快速收藏/取消
- 批量管理
- 标签分类
- 搜索过滤

---

### 4. 专项复习 📚
**功能描述**：针对薄弱环节进行强化训练

#### API 接口
- `GET /api/review/topics` - 获取复习主题列表
- `POST /api/review/start` - 开始复习
- `GET /api/review/record` - 获取复习记录
- `GET /api/review/report` - 生成复习报告

#### 复习模式
- **章节复习** - 按章节顺序练习
- **难度复习** - 专攻高难度题目
- **高频错题** - 错误率最高的题目
- **考前冲刺** - 精选高频考题
- **弱项强化** - 根据答题数据分析

---

### 5. 模拟考试 🎯
**功能描述**：全真模拟考试环境，限时答题

#### API 接口
- `GET /api/exam/papers` - 获取试卷列表
- `POST /api/exam/start` - 开始考试
- `POST /api/exam/submit` - 提交试卷
- `GET /api/exam/result/{examId}` - 查看成绩
- `GET /api/exam/history` - 考试历史
- `GET /api/exam/analysis/{examId}` - 试题分析

#### 考试规则
- **考试时间**：45 分钟
- **题目数量**：100 题
- **题型分布**：单选 80 题 + 判断 20 题
- **及格分数**：90 分
- **自动交卷**：时间到自动提交
- **即时评分**：提交后立即显示成绩

#### 功能特性
- 倒计时提醒
- 答题卡导航
- 题目跳转
- 标记不确定题目
- 成绩趋势图

---

### 6. 题库练习 🔖
**功能描述**：完整的题库，支持多种练习模式

#### API 接口
- `GET /api/question/categories` - 获取章节分类
- `GET /api/question/list` - 题目列表（分页）
- `GET /api/question/detail/{id}` - 题目详情
- `POST /api/question/answer` - 提交答案
- `GET /api/question/random` - 随机抽题

#### 练习模式
- **顺序练习** - 按题号顺序
- **随机练习** - 随机抽取题目
- **章节练习** - 选择特定章节
- **难度练习** - 按难度等级

#### 功能特性
- 答题解析立即显示
- 收藏/错题自动记录
- 答题进度保存
- 快捷键支持（1-4 选项）

## 🗄️ 数据库表设计

### 新增表结构

#### user_collection - 用户收藏表
```sql
CREATE TABLE user_collection (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  question_id BIGINT NOT NULL,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  deleted INT DEFAULT 0,
  UNIQUE KEY uk_user_question (user_id, question_id)
);
```

#### driving_news - 驾考资讯表
```sql
CREATE TABLE driving_news (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  title VARCHAR(200) NOT NULL,
  content TEXT NOT NULL,
  cover_image VARCHAR(500),
  category INT DEFAULT 1,
  view_count INT DEFAULT 0,
  publish_time DATETIME,
  author VARCHAR(50),
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  deleted INT DEFAULT 0
);
```

#### system_config - 系统配置表
```sql
CREATE TABLE system_config (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  config_key VARCHAR(100) NOT NULL UNIQUE,
  config_value TEXT,
  config_name VARCHAR(100),
  remark VARCHAR(500),
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  deleted INT DEFAULT 0
);
```

## 💡 技术实现建议

### 后端
- 使用 MyBatis-Plus 的 QueryWrapper 实现复杂查询
- Redis 缓存热门题目和资讯
- 定时任务统计学习数据
- WebSocket 实现考试倒计时同步

### 前端
- Pinia 统一管理用户状态
- Vue Router 路由守卫控制权限
- Axios 拦截器统一处理错误
- ECharts 数据可视化
- LocalStorage 缓存答题进度
